#### To Build and Run:

cd kafka-[consumer/producer]
mvn clean package
java -jar target/kafka-producer-1.0-SNAPSHOT-jar-with-dependencies.jar


#### Export and Run:
mvn clean compile assembly:single
java -jar kafka-producer-1.0-SNAPSHOT-jar-with-dependencies.jar
